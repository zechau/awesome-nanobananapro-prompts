### **I. Core Features Overview**

* **Resolution Support**: Outputs up to 4K resolution images
* **Multi-round Editing**: Supports conversational, multi-round image editing workflows
* **Multi-image Composition**: Combines up to 14 input images into 1 output image
* **Search Enhancement**: Integrated Google search capabilities for more precise, up-to-date knowledge support
* **Deep Thinking & Logical Reasoning**: Performs physics simulation and logical reasoning before image generation, not just simple imitation
* **Cross-modal & Multilingual**: Supports generating text in multiple languages, one-click translation and localization
* **Ultra-long Context**: 64K input token limit, understands extremely long text prompts
* **Character Consistency**: Maintains appearance consistency of up to 5 characters across multiple images
* **Professional Creative Control**: Adjust camera angles, lighting, styles, color grading through text instructions
* **AI Transparency**: All generated content includes SynthID digital watermarks for source verification

You are an expert Information Designer. Your goal is to extract the essential structure from a web page to create a clear, educational infographic.

Analyze the content at this URL: **${url}**

TARGET LANGUAGE: **${language}**.

Provide a structured breakdown specifically designed for visual representation in **${language}**:

1. INFOGRAPHIC HEADLINE: The core topic in 5 words or less (in **${language}**).
2. KEY TAKEAWAYS: The 3 to 5 most important distinct points, steps, or facts (in **${language}**). THESE WILL BE THE MAIN SECTIONS OF THE IMAGE.
3. SUPPORTING DATA: Any specific numbers, percentages, or very short quotes that add credibility.
4. VISUAL METAPHOR IDEA: Suggest ONE simple visual concept that best fits this content (e.g., "a roadmap with milestones", "a funnel", "three contrasting pillars", "a circular flowchart").

Keep the output concise and focused purely on what should be ON the infographic. Ensure all content is in **${language}.**

After getting this article summary, we can start image generation.

Create a professional, high-quality educational infographic based strictly on this structured content plan:

**${structuralSummary}**

VISUAL DESIGN RULES:

\- **${styleGuidelines}**

\- LANGUAGE: The text within the infographic MUST be written in **${language}**.

\- LAYOUT: MUST follow the "VISUAL METAPHOR IDEA" from the plan above if one was provided.

\- TYPOGRAPHY: Clean, highly readable sans-serif fonts. The "INFOGRAPHIC HEADLINE" must be prominent at the top.

\- CONTENT: Use the actual text from "KEY TAKEAWAYS" in the image. Do not use placeholder text like Lorem Ipsum.

\- GOAL: The image must be informative and readable as a standalone graphic.

#### **Official Intervention: 7 Image Generation Tips**

Besides using these already-written prompts, Google has painstakingly provided detailed Nano Banana Pro usage techniques, partly because they're afraid we won't operate properly, leading to misunderstandings about the model. On the other hand, Google really hopes Nano Banana Pro won't just be a fun image generation model - they truly expect it to improve our productivity.

The importance of prompts for getting precise content control, besides the model's own capabilities, the remaining determining factor is the prompts themselves. Google mentions again that **a good prompt should clearly include subject, composition, action, location, style, and editing instructions.**

Briefly describe the image subject with concise language, for example, specifically explaining what people or things are in the picture? For composition, tell the model how the photo is framed? Is it close-up, wide-angle lens, low-angle shot, or portrait lens, etc.

![Image](https://maynor123-1301929665.cos.ap-guangzhou.myqcloud.com/202511211815961.jpeg)

Example: A fluffy calico cat wearing a small wizard hat, casting magic on a sunlit grassland at sunset, low-angle shot, photorealism

Next is describing the action, what's happening to the subject in the picture? And where it's happening, describing the specific scene. Finally, determine the photo style we want, for example, 3D animation, film noir, watercolor, photorealism, 1990s product photography, etc., different overall aesthetic style choices.

**If pursuing more controllable image generation effects, Google also mentioned further improving camera, lighting, and format details to refine prompt content.**

For example, current Nano Banana Pro supports custom composition and aspect ratios. We can choose 9:16 vertical posters, 21:9 cinematic wide screens, and other screen formats.

Additionally, in photography and lighting details, Google's reference prompt is to guide shooting like a cinematographer, with terminology as professional as possible, for example, "low-angle shot, shallow depth of field (f/1.8), golden hour backlight creating long shadows, cinematic color grading with soft blue-green tones"; and other prompt language.

![Image](https://maynor123-1301929665.cos.ap-guangzhou.myqcloud.com/202511211815429.jpeg)

Example: A fluffy calico cat wearing a small wizard hat, casting magic on a sunlit grassland at sunset, low-angle shot, shallow depth of field (f/1.8), golden hour backlight creating long shadows, cinematic color grading with soft blue-green tones. Photorealism. 21:9 cinematic widescreen. Title "Cat Magic Casting" displayed at top in cute rainbow font.

When handling text processing tasks, need to clearly specify text content and display position, and corresponding fonts. For chart infographics, emphasize accuracy requirements in prompts and ensure input is correct.

When involving multi-image reference and image editing, need to clearly tell the model each image's role and specific editing needs.

### **IV. Official Prompt Engineering Guidelines**

Google emphasizes the importance of **good prompts** that clearly include **subject, composition, action, location, style, and editing instructions**.

**Basic prompt structure**: Briefly describe the image subject, composition (close-up, wide-angle, low-angle shot, portrait shot, etc.), action, scene location, and desired style (3D animation, film noir, watercolor, photorealism, 1990s product photography, etc.).

**Advanced controls**: For more precise generation, enhance camera, lighting, and format details:

- Custom aspect ratios (9:16 portrait posters, 21:9 cinematic widescreen)
- Professional photography terminology: "Low-angle shot, shallow depth of field (f/1.8), golden hour backlight creating long shadows, cinematic color grading with soft blue-green tones"
- Text processing: Specify text content, display position, and fonts
- Charts and infographics: Emphasize accuracy requirements
- Multi-image references: Clearly specify each image's role and editing requirements

#### **7 Key Application Scenarios**
